# Monique Bakery Website

Live Demo: https://abraham001-eng.github.io/monique-bakery/

Responsive bakery landing page built for a Kano-based client.

## Tech Stack
- HTML5
- CSS3 - Flexbox, Grid, Animations, Media Queries
- Vanilla JavaScript - DOM, Intersection Observer, Scroll effects

## Features
- Fully responsive design - mobile, tablet, desktop
- Fixed navbar with scroll effects
- CSS animations & parallax hero section
- Interactive cake gallery & testimonials slider
- Contact form UI
